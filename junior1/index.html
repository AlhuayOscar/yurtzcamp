Maqueta un login con Tailwind CSS

Usa este archivo como punto de partida. La idea es que tu companero complete las clases de Tailwind para lograr un login centrado, con un logo arriba, labels a la izquierda, inputs a la derecha y un boton de ingreso bien visible.

Checklist

  • Agregar un fondo suave que ocupe toda la pantalla.
  • Centrar la tarjeta de login vertical y horizontalmente.
  • Usar bordes redondeados y sombra en la tarjeta.
  • Dejar el logo como primer elemento visual.
  • Ordenar cada fila con label a un lado e input al otro.
  • Estilizar el boton "Ingresar" con colores definidos.
